# LogicLand — Branding

**Tagline:** Think Clearly. Build Boldly. Create Fearlessly.

## Colour (tokens in apps/web/tailwind.config.ts)
| Token | Hex | Use |
|---|---|---|
| Wonder Violet (`brand`) | #6C5CE7 | primary, identity |
| Soft Violet (`brand.soft`) | #A29BFE | surfaces, accents |
| Encouragement Gold (`sunburst`) | #FFB020 | rewards, XP, celebration |
| Growth Green (`meadow`) | #22C55E | progress, success |
| Curiosity Blue (`sky`) | #38BDF8 | info, playful highlights |

Bright colours used **with restraint** on calm, warm neutrals. Full light/dark
tokens live in `apps/web/src/app/globals.css`.

## Typography
- **Display:** Baloo 2 (rounded, friendly) — headings, hero.
- **Body:** Nunito (readable, warm) — everything else.
- Large sizes, short lines, high contrast.

## Illustration
Friendly, rounded characters led by **Robo**. Soft gradients, gentle shadows,
inclusive and diverse. Never scary, never busy.

## Voice
Warm, encouraging, playful, clear. Speaks *with* children, never down to them.
Parents: reassuring and specific. Teachers: capable and efficient.
