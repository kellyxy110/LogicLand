# LogicLand — Roadmap (Four Phases)

## Phase 1 — Foundation & Core Classroom (current)
Monorepo, docs, architecture, schema, engine skeleton, web skeleton, CI/Docker,
12-week curriculum data, AI Helper + generators wired as services. Definition of
done: structure + docs + standards reviewed. *No feature build before review.*

## Phase 2 — The Living Classroom
Full student mission flow with an activity runtime, real gamification (XP/coins/
streaks/unlocks + reward animations), teacher builders & generators end-to-end,
parent dashboard with weekly reports and messaging, badges & certificates issued.

## Phase 3 — Intelligence & Adaptivity
Adaptive revision suggestions, richer analytics & mastery modelling, project
review workflows, lesson recaps, parent growth-timeline narratives, notification
system, storage-backed media.

## Phase 4 — Academy Scale
Multi-classroom / multi-teacher, admin role, organization scoping, marketplace-
ready curriculum authoring, future AI (voice tutor, speech, image generation),
mobile-app polish, observability & cost controls at scale.
